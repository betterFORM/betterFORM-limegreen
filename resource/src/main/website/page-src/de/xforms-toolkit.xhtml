XForms Toolkit

zukunftsorientierte eForms in Ihrem Browser.

Umfassende W3C XForms 1.1 Umgebung

Das XForms Toolkit ist eine der fortschrittlichsten XForms Implementierungen für dynamische, formular-basierte Anwendungen, die in jedem Browser ohne Plugins laufen.

Das betterFORM XForms Toolkit läuft auf Windows, Mac OS X, Linux, FreeBSD and weiteren Betriebssystemen.

Flexibel und leicht zu integrieren

Das XForms Toolkit passt sich leicht in existierende Infrastrukturen ein und bietet vielfältige Schnittstellen zur Anbindung von kundenspezifischer Logik.

Durch die direkte Unterstützung von ReST, SOAP, XML-RPC und HTTP ist es der ideale Partner für SOA und EAI Anwendungen.

betterFORM-basierte Anwendungen eignen sich hervorragend für agile Vorgehensweisen und sind flexibel auch an zukünftige Anforderungen anpassbar, wobei die Komplexität der zugrundeliegenden Technologien verborgen wird.

Basiert durchgehend auf Open Source und Open Standards

Das XForms Toolkit basiert komplett auf freier Open Source Software und setzt ausschließlich zukunftssichere Offene Standards ein. Dank der wohlbekannten BSD Lizenz kann das Projekt auch im kommerziellen Umfeld kostenlos eingesetzt werden.

Produktivität erhöhen - Kosten senken

Das XForms Toolkit bietet eine Abstrationsschicht für Ihre Webanwendung und hebt Ihre Produktivität auf eine höhere Stufe.

Der deskriptive und generische Ansatz macht wiederholtes Scripting überflüssig und senkt so die Entwicklungskosten. Sie erhalten fertige Anwendungen innerhalb weniger Tage.

Attraktive, dynamische und konsistente Benutzeroberflächen für Ihre Daten

betterFORM wandelt XForms in hochmoderne Web 2.0 Anwendungen, die sowohl auf dem Desktop als auch auf mobilen Geräten ausgeführt werden können. Dank der vielseitigen Layoutmöglichkeiten können Sie das Look&Feel zielgerichtet auf Ihre CI anpassen.

XForms-basierte Anwendungen ermöglichen es, Daten aus verschiedensten Quellen zu aggregieren und auf diesen zu operieren. Ein umfassende Bibliothek von 'XForms Aktionen' vereinfach sowohl Daten- als auch User Interface-Manipulationen ohne eine einzige Zeile Script Code zu schreiben.

Erfassen, Pflegen und Präsentieren Sie ihre Daten

Datenvalidität ist für jede ernsthafte Anwendung essentiell. XForms bietet feingranulare Validierungen für Ihre Daten und unterstützt Nutzer bei der Datenerfassung mit frühzeitigen Rückmeldungen und voll dynamischen Benutzeroberflächen.

XForms ist deklarativ und kann ganze Anwendungen durch Generierung aus Schemadaten für Industrie Standards erzeugen.
