Software

betterFORM bietet ein ausgereiftes Open Source Software Paket unter der liberalen BSD Lizenz an. Dieses Paket kann ohne Lizenzkosten genutzt, modifiziert, adaptiert, erweitert, weitervertrieben und mit eigenen Applikationen kombiniert werden.

betterFORM Project

Seit mehr als 10 Jahren entwickelt das betterFORM Project Open Source XForms Software.

Die Kernkomponente unseres Softwarepakets besteht aus unserer W3C XForms 1.1 Implementierung, die in vielen verschiedenen Szenarien und Architekturen zum Einsatz kommen kann:

Alle Komponenten von betterFORM werden über einen einfach zu nutzenden grafischen Installer angeboten.

  • eingebettet in eine Java Applikation oder ein Applet
  • eng integriert mit einer Webapplikation
  • als eigenständige Webapplikation
  • als eigenständiger Server

XForms Toolkit

Das XForms Toolkit ist eine Implementierung des W3C XForms 1.1 Standards und läuft ohne ein Plugin auf allen gängigen Browsern.

Das XForms Toolkit hilft bei:

  • der schnellen Entwicklung von fortgeschrittenen Front-end Applikationen
  • komplexen Formular- (und Subformular) Anwendungen
  • aufwendigen Validierungen, Berechnungen und Abhängigkeits-Management
  • Benutzeroberflächen für Webservices
  • Editoren für XML Datenformate
  • der Entwicklung von cross-browser Anwendungen
  • der Unterstützung von XForms auf allen gängigen Desktop- und Mobil-Browsern
  • der massiven Reduktion von Skriptcode

betterFORM XML Suite

Die betterFORM XML Suite setzt auf dem XForms Toolkit auf und integriert es eng mit der eXist XML Datenbank. Zusammen sind beide Produkte mehr als die Summe ihrer Teile - es können reine XML Anwendungen mit Hilfe von XForms, XQuery, XPath, XSLT, XInclude und anderen offenen Standards entwickelt werden.

Diese Architektur ist unter dem Akronym 'XRX' bekannt geworden. Dies steht für 'XForms' 'ReST' und 'XQuery' und beschreibt eine vollständige auf XML basierende Umgebung, die zur Entwicklung komplexer Anwendungen eingesetzt werden kann. Die Anwendung wird vollständig in der Datenbank abgelegt, was die Aktualisierung und Pflege erleichtert.

Ein bemerkenswerter Produktivitäts-Gewinn resultiert aus dem Wegfallen jeglicher Datenkonvertierung, da die Daten zu jeder Zeit in XML vorliegen und mit XML Werkzeugen bearbeitet werden. Die resultierende Anwendung ist einfach zu installieren, verteilen und erweitern.
